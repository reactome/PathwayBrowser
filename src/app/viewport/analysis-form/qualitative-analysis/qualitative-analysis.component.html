Qualitative Entity Enrichment Analysis
Enrichment analysis is a statistical test that determines whether certain Reactome pathways are over-represented (enriched) in the submitted data. It answers the question ‘Does my list contain more proteins for pathway X than would be expected by chance?’
You can provide a simple list of entities (genes, proteins, chemicals, ...) of interest to see which pathways are enriched using this method, but you can also provide numerical data associated to those names so that the enriched pathways are coloured according to their average value.

Options

Project to Human

All non-human identifiers are converted to their human equivalents

Explanations

Every species has its own set of identifiers (genes, proteins, chemicals, mRNA, etc...)

Reactome curates human pathways and infers their existence in other species using orthology information from the PANTHER knowledge base

When this option is selected, all non-human identifiers in your sample are mapped to their human equivalents before the analysis is performed

@if (projectToHuman()) { check }

Include Interactors

IntAct interactors are used to increase the analysis background

Explanations

Selecting this option will take interactors into account when performing the analysis.

Only gene symbols, chemical names and Uniprot or ChEBI identifiers can be used to identify interactors.

Curated entities and interactors backgrounds may overlap (e.g. entities can interact with themselves (dimers) or with other entities in the diagram).

The analysis result uses the union of both backgrounds. Entities in the overlapping area are only counted once.

@if (includeInteractors()) { check }
